# Aletheia: Digital Petri Dish

An autonomous multi-agent simulation lab for observing emergent LLM behaviors, dialogue loops, and system dynamics in a controlled "petri dish" environment. Configure agents, set a genesis prompt, and watch the simulation unfold with real-time stats and persona controls.

## What it does

- Orchestrates multiple AI personas that take turns responding to a shared prompt.
- Lets you customize each agent's model, system prompt, temperature, and token limits.
- Streams responses with live telemetry (latency, tokens, cost estimate).
- Includes a built-in configuration console for personas, genesis prompt, and constraints.
- Ships as a Progressive Web App (PWA) with offline support and install prompts.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set `VITE_GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3. Run the app:
   `npm run dev`

## PWA Notes

- The app registers a service worker for offline support.
- Use the in-app install prompt to add the app to your home screen.
